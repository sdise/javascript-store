#                            telegram web k 去除表情包  

代码由AI提供  
### 脚本逻辑解析：

1.  **彻底消失 (`display: none`)**：
    *   之前的版本使用 `opacity: 0`，虽然看不见，但表情包原本占用的高度还在，会导致聊天记录出现大片空白。
    *   现在使用 `display: none`，浏览器在渲染时会直接跳过这些消息，**后续的消息会自动向上贴合**，就像这些表情包从未被发送过一样。

2.  **消息级屏蔽 (`.bubble:has(...)`)**：
    *   它不是只隐藏图片，而是寻找“内部含有表情包”的**整条消息气泡 (`.bubble`)**。
    *   这样可以连带隐藏掉气泡的边框、阴影以及表情包下方可能存在的空隙。

3.  **极速性能**：
    *   去掉了所有的 `EventListener`（事件监听）和 `MutationObserver`（DOM 监控）。
    *   CSS `:has()` 是由浏览器 C++ 内核直接处理的，比任何 JS 脚本都要快。当新消息进入时，浏览器会自动根据样式表决定是否渲染，不会产生卡顿感。

4.  **精准排除**：
    *   因为作用域限定在 `.bubble`（消息气泡）内，所以**侧边栏联系人、对话框顶部的用户昵称**中的 Emoji 装饰不会受到任何影响，它们会照常显示。
    *   普通发送的照片（`.media-photo`）会被强制保留显示，确保不会误删正常的图片交流。

### 注意事项：
*   **不可恢复**：这个版本去掉了右键功能，这意味着一旦开启，你将完全看不到任何表情包消息。
*   **浏览器要求**：`:has()` 选择器需要较新版本的浏览器（Chrome 105+, Edge 105+, Firefox 103+, Safari 15.4+）。如果你的浏览器版本太旧，脚本将不会生效。
